Instrucciones a seguir para ejectuar el programa desde la caja de comandos:

1. Abrir el cmd o una caja de comandos equivalente.

2. Una vez abierto, hemos de introducir "java" más el path donde se encuentra el programa en cuestión,
   como por ejemplo:
	"D:\Sprints\src\main\java\Sprint1\Tasca_S1_05\Nivell1\Exercici1\Main.java"

3. Con esto el programa debería funcionar, en este caso primero nos pedirá un path que queremos ordenar.
	Seguido de el path de un txt que queremos que nos lea.
